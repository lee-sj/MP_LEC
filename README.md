# 마이크로프로세서 및 실습 

## 실습 코드 저장소

* lab_N 으로 lab 폴더 생성
* img 폴더의 labN으로 lab_N 관련 실행 이미지 및 실행 영상 저장
* 이미지 및 영상은 파일명 규칙 없음 

## 실습 진행

워크스페이스는 매번 다시 만들어서 시작  

1. lab 폴더 생성 - 네이밍 : lab_x  
2. 프로젝트 폴더생성 - 네이밍 : labx-x 
3. File > New > workspace  
4. project > Create New Project - 네이밍 : labx-x
